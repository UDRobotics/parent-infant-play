This is a simple Python script for uploading images to Google Gemini for analysis.  It was created to generate the computer vision (CV) data referred to in the paper (under review) "Parents Use Different Positioning Practices with Infants During Researcher-Observed Play
Sessions Than They Do Throughout the Day," by Julie M. Orlando, Abram Banoub, Samantha Namit, Eman Alhindi, Amani Kiruga, Christopher Rasmussen, and Michele A. Lobo.

Instructions for installation and running are in comments in the code.
